Polymorphism :-
    Defination:- Polymorphism means the same method can act differetly depending on runing object.

Types of Polymorphism:-
1) Runtime Polymorphism (method overriding) :- 
    1.runtime Polymorphism occurs when method call is resolved at runtime based on object.
    2. using inheritance and use @override notation


2) Compile time Polymorphism (method overloding) :-
    1.Compile time Polymorphism with achive using same method name but different signature(parameters,types of parameters,number of parameters).
