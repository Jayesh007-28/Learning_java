Inheritance :- 
    1.Inheritance allows child class(derived class) to reuse the properties and method of parent class(base class).
    2.Instead of rewrinting the same code in multiple palces we allow child class to reuse the parent properties and methods.

Types of Inheritance:-
1.Single Inheritance
2.Muli-level Inheritance
3.Hierarchical Inheritance
4.Multiple Inheritance (not suppored by java but we can achive this by interface)

1.Single Inheritance
Benefits :- 1) reusability 2) maintainability 3)scalability

* syntax of Single Inheritance :-

    class parent {
        // properties and methods
    }
    class child extends parent {
        // additional properties and methods
    }

4.Multiple Inheritance (Using Interface) :-
    Multiple inheritance is a concept where one class inherits properties and behavior from more than one parent class.

        Class A

        --------------
        |            |
     Class B      Class C
        |            |

             |
          Class D

          Example :- We can take Two interfaces one is account operation interface which content deposit functionallty other is loan operation interface which content loan functionallty then samrt account class implements both the interfae.

Rules of inheritance :- 
1)Parent construtor is always calls first.
2)Parent cannot access child only members.
3)Parent referrals can point to child object.
4)Multiple inheritance is not suppored by class but we can use interface to implements multiple     inheritance.
